# HTML_Practice
HTML Practice Projects

Overview

This repository contains HTML projects that I built while learning the fundamentals of web development. These projects helped me understand the core concepts of HTML, including page structure, navigation, forms, tables, images, and hyperlinks.

Projects Included

1. Event Manager

A simple Event management webpage created using HTML.

Features:
- practice with basic html tags and hyperlinks
- Practice with forms and input elements
- Structured page layout using HTML

2. MS Dhoni Fan Page

A fan page dedicated to MS Dhoni, one of India's greatest cricketers.

Features:

- Images and biography section
- Career achievements and statistics
- External links
- Fan registration form
- Navigation between multiple HTML pages

HTML Concepts Practiced

- HTML Boilerplate
- Headings ("h1" - "h6")
- Paragraphs ("p")
- Hyperlinks ("a")
- Images ("img")
- Lists ("ul", "ol", "li")
- Tables ("table", "tr", "th", "td")
- Forms ("form")
- Input Fields
- Radio Buttons
- Checkboxes
- Dropdown Menus ("select")
- Text Areas ("textarea")
- Multi-page Navigation
